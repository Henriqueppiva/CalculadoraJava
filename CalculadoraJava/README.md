# Calculadora em Java

Projeto simples desenvolvido em Java para praticar conceitos básicos de programação.

## Objetivo

Praticar lógica de programação e organização de código utilizando Java.

## Funcionalidades

A calculadora permite realizar as seguintes operações:

- Soma
- Subtração
- Multiplicação
- Divisão

O sistema funciona através do terminal (console), onde o usuário informa dois números e escolhe a operação desejada.

## Tecnologias utilizadas

- Java

## Como executar

1. Clone o repositório:

git clone https://github.com/seuusuario/calculadora-java

2. Compile os arquivos:

javac Main.java Calculadora.java

3. Execute o programa:

java Main
